Wonderful Editor は Qiita風の記事作成アプリ。Ruby, Rails のアウトプットのために作成しました
# 主な機能
 - 記事の投稿機能
 - ユーザー登録・サインイン/サインアウト
 - 記事の下書き機能
 - マイページ（自分が書いた記事の一覧）
 - 記事の下書き機能
 
 # 技術
 - Rails6.0.3.7
 - Ruby2.7.2
 - Node14.20.0
 - Vue.js2.6.11
 - DB: PostgreSQL
 - RSpec
 
 # 工夫
